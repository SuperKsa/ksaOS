KsaOS 1.0
===============


##环境要求：
~~~
PHP7.0+ / Mysql 5.7 / Redis
最低硬件：2核4G 硬盘10G+
~~~


##安装注意:
~~~
APP为项目文件夹
ksaOS为底层框架，请勿修改
~~~
>请先修改[_config_demo.php](_config_demo.php)参数后,将文件改名为config.php
